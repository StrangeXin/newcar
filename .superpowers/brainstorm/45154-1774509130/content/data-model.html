数据模型设计

套餐订阅 + AI 用量追踪的核心表结构

1. 套餐订阅层（面向用户）

SubscriptionPlan（套餐定义）
id — String @id
name — String (FREE / PRO / PREMIUM)
displayName — String ("免费版" / "Pro" / "Premium")
price — Int (分，0 = 免费)
billingCycle — String (MONTHLY / YEARLY)
dailyConversationLimit — Int (每日对话数上限)
monthlyTokenLimit — Int (每月 token 上限)
features — Json (功能开关列表)
modelAccess — String[] (可用 AI 模型)
sortOrder — Int
isActive — Boolean
📌 运营可配置，不硬编码
UserSubscription（用户订阅状态）
id — String @id
userId — String (→ User)
planId — String (→ SubscriptionPlan)
status — String (ACTIVE / EXPIRED / CANCELLED)
startedAt — DateTime
expiresAt — DateTime?
dailyConversationsUsed — Int
dailyResetAt — DateTime
monthlyTokensUsed — Int
monthlyResetAt — DateTime
source — String (SYSTEM / ADMIN / PAYMENT)
📌 每用户一条 ACTIVE 记录

2. AI 用量追踪层（面向运营）

AiUsageLog（请求级日志）
id — String @id
userId — String (→ User)
conversationId — String (→ Conversation)
model — String ("claude-sonnet-4-6")
inputTokens — Int
outputTokens — Int
cacheReadTokens — Int?
cacheWriteTokens — Int?
estimatedCostUsd — Float
requestType — String (CHAT / TOOL_CALL / SNAPSHOT)
durationMs — Int?
createdAt — DateTime
📌 每次 API 请求一条记录
AiConversationUsage（对话级汇总）
id — String @id
conversationId — String @unique (→ Conversation)
userId — String (→ User)
totalInputTokens — Int
totalOutputTokens — Int
totalCostUsd — Float
requestCount — Int
primaryModel — String
createdAt — DateTime
updatedAt — DateTime
📌 每次对话结束/更新时聚合写入

3. 关系概览

Entity Relationship
User 1 ← → 1 UserSubscription N ← → 1 SubscriptionPlan
│
User 1 ← → N AiUsageLog N ← → 1 Conversation
│
Conversation 1 ← → 1 AiConversationUsage
