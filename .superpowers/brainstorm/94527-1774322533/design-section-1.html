3 · 主题切换架构

html[data-theme="orange"] — 默认橙暖主题 CSS 变量
html[data-theme="indigo"] — 靛蓝极简覆盖层
localStorage("theme") — 用户偏好持久化，页面刷新不闪烁
useTheme() hook — 全局状态，所有组件通过 CSS 变量响应，无需 React re-render
💡 新增主题只需添加一个 CSS 规则块，不改任何组件代码
